# Password Strength Checker

A simple Python script to check the **strength of your password** based on length, uppercase letters, lowercase letters, numbers, and special characters.



## Features

- Checks if the password has:
  - Minimum length of 8 characters
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Outputs a **strength score (0-100%)**.


## Requirements

- Python 3.x

No external libraries required.


## Usage

1. Run the script:

**bash**
python password_checker.py

2. Enter your password when prompted:
3. The script will display your password strength and any missing requirements
